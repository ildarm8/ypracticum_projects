# Yandex Pricticum Projects

Данные проекты были выполнены в ходе обучения в Яндекс.Практикуме, профессии "Аналитик данных".

| Название проекта | Описание | Используемые библиотеки | 
| :---------------------- | :---------------------- | :---------------------- |
| [Исследование надёжности заёмщиков банка](PRJ1-Предобработка данных) | Проект посвящен предобработке данных. В ходе работы были обработаны некорректные, пропущенные значения, созданы категории в данных, в том числе и на основе стемминга строк, обработаны дубликаты, подготовлены графики.| *pandas* |

| [Продажа квартир в Санкт-Петербурге — анализ рынка недвижимости](PRJ2-Исследовательский анализ данных) | Проект посвящен исследовательскому анализу данных. На основе данных сервиса Яндекс.Недвижимость определена рыночная стоимость объектов недвижимости разного типа, типичные параметры квартир, в зависимости от удаленности от центра. Проведена предобработка данных. Добавлены новые данные.Построены гистограммы, боксплоты, диаграммы рассеивания.| *pandas,matplotlib* |
